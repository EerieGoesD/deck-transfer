# Deck Transfer

A plug-and-play desktop app for transferring files from your PC to a Steam Deck over an ethernet cable.

## Why?

Transferring files to a Steam Deck over WiFi is slow, and forcing the connection through ethernet instead of WiFi requires manual network configuration that most users shouldn't have to deal with. Deck Transfer handles all of that automatically.

## Features (Planned)

- Auto-detect Steam Deck connected via ethernet
- Force file transfers over the ethernet link (not WiFi)
- Simple drag-and-drop file transfer UI
- Transfer progress with speed indicator
- No manual network configuration required
